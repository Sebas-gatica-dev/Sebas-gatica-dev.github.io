Nodejs

Ideado como un entorno de ejecución de JavaScript orientado a eventos asíncronos, Node.js está diseñado para crear aplicaciones network escalables. El alta gamma de herramientas que ofrece nodejs, a través de su gestor de paquetes npm (Node Package Manager), agilizan enormemente el trabajo, a partir del framework Express, se combinan todo tipo de herramientas que facilitan en gran medida el desarrollo backend.

SQL

SQL (Structured Query Language) es un lenguaje de dominio específico, diseñado para administrar, y recuperar información de sistemas de gestión de bases de datos relacionales. El diseño de tablas, de llaves primarias y foráneas, puede realizarse a través de MySQL WorkBench, generando código SQL exportable. Las transacciones que se realizan, por las peticiones a la base de datos, son de tipo ACID (el acrónimo para los conceptos Atomicity, Consistency, Isolation y Durability, que en español significa, respectivamente, atomicidad, consistencia, aislamiento en las transacciones y durabilidad).
Con la utilización de la biblioteca Bcript, se puede asegurar la encriptación de contraseñas en la base de datos, así como de cualquier información sensible. Además, otros paquetes de npm nos permiten mejorar la seguridad, evitando así, por ejemplo, los ataques de tablas arcoíris.

Express

Express es el framework web más popular de Node, en torno a express hay librerías para trabajar con cookies, sesiones, inicios de sesión de usuario, parámetros URL, datos POST, cabeceras de seguridad y muchos más. En mis proyectos doy uso a varias bibliotecas, que pueden combinarse a express, como express-validator, express-session, express-generator, Bcript, JsonWebToken, express-ejs-layout, entre otras tecnologías, que no solo agilizan el desarrollo, sino que brinda seguridad y vuelven la aplicación dinámica escalable. Además de contar con una gran comunidad, dentro de la cual es sencillo sacarse las dudas, y encontrar de la mano de otros desarrolladores, mejores prácticas y formas de aplicación más eficientes para estas aplicaciones, todo esto junto con la clara documentación brindada por npm. Mi perfil como desarrollador está basado, en la facilidad para adaptación y la rápida comprensión de documentación.
