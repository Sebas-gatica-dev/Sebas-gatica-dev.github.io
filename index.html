SQL

SQL (Structured Query Language) is a domain-specific language designed to administer and retrieve information from relational database management systems. The design of tables, of primary and foreign keys, can be done through MySQL WorkBench, generating exportable SQL code. The transactions that are carried out, by the requests to the database, They are of type ACID (the acronym for the concepts Atomicity, Consistency, Isolation and Durability, which in Spanish means, respectively, atomicity, consistency, isolation in transactions and durability).
With the use of the Bcript library, you can ensure the encryption of passwords in the database, as well as any sensitive information. In addition, other npm packages allow us to improve security, thus avoiding, for example, rainbow table attacks.

Express

Express is the most popular Node web framework, around express there are libraries to work with cookies, sessions, user logins, URL parameters, POST data, security headers and many more. In my projects I use various libraries, which can be combined with express, such as express-validator, express-session, express-generator, Bcript, JsonWebToken, express-ejs-layout, among other technologies, which not only speed up development, but also that provides security and makes the application dynamic scalable.
