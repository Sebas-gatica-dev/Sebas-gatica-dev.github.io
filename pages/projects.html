¡Bienvenido a la sección de mis proyectos!, las características son explicadas a fondo en los repositorios, ¡te invito a visitarlos!

Los proyectos aquí presentes buscan dejar en evidencia mis conocimientos como desarrollador full stack junior, interfaces simples y prolijas, junto con un backend sencillo y funcional.

Podrá encontrar que los CRUD están realizados en dos tipos de bases de datos diferentes, de tipo relacional, y base de datos en la nube. Además, dos de los proyectos están enfocados en la utilización de APIs externas.

ToDo-App-React-Express-Bootstrap

Esta aplicación ToDO, no es más que una interfaz de anotación de tareas pendientes, a medida que se agregan las tareas, se visualizaran como una nueva fila de la tabla, las mismas pueden modificarse y eliminarse, el objetivo de este proyecto, es ilustrar de manera sencilla el funcionamiento basico de una aplicacion CRUD. Mira un video ilustrativo

El front de la aplicación está basado en React. Las clases de Bootstrap 5 son utilizadas para los estilos de los componentes, así como la funcionalidad de grilla dinámica que nos ofrece dicho framework. Ademas axios nos permitira tomar lo que necesitemos tomar del backend, para asi mapearlo en la tabla.

El back-end esta trabajado con el framework Express, utilizando el middleware cors, a través de sequelize, se establece una conexión a una base de datos relacional, se instancia el modelo de la tabla y las rutas correspondientes, y se crean las diferentes funciones CRUD, a través de las cuales se accede a la respectiva tabla.
¡Visita el repositorio!

CRUD-React-Firebase

Esta aplicación, es un crud de productos, a medida que se agregan los productos y su respectivo stock, se visualizaran como una nueva fila de la tabla, las mismas pueden modificarse y eliminarse, el objetivo de este proyecto, es ilustrar de manera sencilla el funcionamiento básico de una aplicación CRUD. Mira un video ilustrativo

El front de la aplicación está basado en React. Las clases de Bootstrap 5 son utilizadas para los estilos de los componentes, así como la funcionalidad de grilla dinámica que nos ofrece dicho framework.

En este caso el backend es muy sencillo y está integrado al front , utilizando el paquete npm de firebase, a través de la cual se establece una conexión a una base de datos relacional, y se utilizan las diferentes funciones CRUD ya instanciadas en el paquete, que nos ofrece firebase, por medio de las cuales se manipula la información de la nube Firestore, dentro de la cual podemos crear documentos y colecciones.
¡Visita el repositorio!

Crypto-React-Axios

En este proyecto, a través de axios llamamos a la API de coingecko, una plataforma de intercambio y realización de operación con criptomonedas, y además ofrecen servicios gratuitos para acceder a las cotizaciones de criptomonedas. Mira un video ilustrativo

El front de la aplicación esta basado en React. Las clases de Bootstrap 5 son utilizadas para los estilos de los componentes, así como la funcionalidad de grilla dinámica que nos ofrece dicho framework.

No posee un backend como tal, los datos son alojados del lado del servidor , se accede a la api con axios, el endpoint al que se llama es 'https://api.coingecko.com/api/v3/coins' , a partir del JSON que nos devuelve la API, se mapean los datos, en la tabla de Bootstrap.
¡Visita el repositorio!

Buscador de imagenes en react

Esta aplicación es un buscador de imágenes que consume la Api de la pagina unsplash, una pagina similar a Pinterest , se genera una filtro de búsqueda, a través de la cual se visualizaran resultados en relación a los escrito en el input de búsqueda. Mira un video ilustrativo

El front de la aplicación esta basado en React. Los estilos fueron trabajados con css puro.

la aplicación no posee un backend como tal, la informacion es trabajada desde el lado del servidor, utilizando Formik, se maneja la informacion que recibe el input, a traves de Fetch se conecta a la api y se aplica el contenido recibido por el el input al final del enpoint, para que la api realice la busqueda.
¡Visita el repositorio!
